GruntJS Setup

1) Installing Node

  • Go to http://nodejs.org/
  • Download & Run the Node installer
  • Open Terminal Window (Unix/Mac) / Run cmd (Windows)

Test Install

$ node --version
v0.8.7
Windows Users If test fails check your system path variable.

2) Installing Grunt

Open Terminal Window (Unix/Mac) / Run cmd (Windows)

npm install -g grunt

Test Install

$ grunt -version
path.existsSync is now called `fs.existsSync`.
grunt v0.3.12

3) Installing Dependencies

GruntJs comes with some buil-in modules but Less is not by default, for this task we are going to use grunt-recess

Open Terminal Window and go to your project folder

$ sudo npm install grunt-recess
...
grunt-recess@0.1.4 node_modules/grunt-recess
└── recess@1.1.6 (colors@0.6.0-1, watch@0.5.1, underscore@1.4.3, less@1.3.3, nopt@2.0.0)
grunt-recess: Grunt task to lint and minify CSS and LESS, using the Twitter RECESS module. More info: https://github.com/sindresorhus/grunt-recess

4) Running Grunt

Open Terminal Window and go to your project folder

$ grunt watch
Window Users You might need to use ./grunt.CMD
